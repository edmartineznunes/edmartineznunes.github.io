Política de Privacidade de Aplicativos

Esta Política de Privacidade descreve como o aplicativo appagtJP, desenvolvido por Eduardo Sidney M. Nunes, coleta, usa e protege informações ao utilizar o aplicativo.

Ao utilizar o aplicativo, você concorda com esta Política de Privacidade.

Coleta e Uso de Informações

O aplicativo utiliza um ID e uma senha fornecidos ao usuário apenas para permitir o acesso ao sistema interno do aplicativo.

Fora essas credenciais de autenticação, o aplicativo não coleta, armazena ou solicita informações pessoais, incluindo, mas não se limitando a:

  • nome
  • endereço de email
  • número de telefone
  • localização
  • contatos
  • fotos ou arquivos
  • dados do dispositivos

Uso de Informações

O ID e a senha são utilizados exclusivamente para autenticação e acesso ao aplicativo.

Essas informações são utilizadas apenas para permitir que o usuário acesse as funcionalidades do aplicativo.

Compartilhamento de informações

O aplicativo não vende, aluga ou compartilha dados pessoais dos usuários com terceiros.

Acesso a informações por terceiros

Este aplicativo não utiliza serviços de terceiros que coletem dados pessoais dos usuários, como plataformas de publicidade ou ferramentas de análise.

Segurança

Adotamos medidas razoáveis de segurança para proteger as informações utilizadas para autenticação contra acesso não autorizado.

Privacidade de crianças

O aplicativo não coleta dados pessoais de usuários, incluindo crianças menores de 13 anos.

Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que os usuários revisem esta página para se manter informados sobre eventuais alterações.
